# MySQL
**数据库的三范式**：

列的原子性不可分割成更多列、

非主键列要完全依赖于主键列、

非主键列之间不存在传递依赖。即非主键列不能依赖于其他非主键列

**范式的发展代表着数据库在结构逻辑上合理化的一个过程，范式的存在有助于减少数据冗余、避免更新异常、简化查询，并提高数据的完整性**
